二、基礎建設：教育帳號與模型配額

教育帳號申請、登入與切換

【申請臺南市教育局專屬 Gmail】

  • 取得一個 xxx@gm.tn.edu.tw 的 Gmail 信箱。
  • 進入 mail.google.com/a/gm.tn.edu.tw。
  • 輸入 OPEN ID 帳號、密碼並同意授權。
  • 若發生網頁錯誤，關掉瀏覽器後再重來。

【申請教育部專屬 Gmail】

  • 取得一個 xxx@go.edu.tw 的 Gmail 信箱。
  • 進入 https://go.k12cc.tw。
  • 點選右上角藍色按鈕「即刻啟用…」。
  • 後續一樣使用 OPEN ID 登入並同意授權。

Gemini 登入

  • 先到 Gmail 用剛剛申請的教育信箱登入。
  • 登入後從右上角「九宮格」點選 Gemini 進入。
  • 或直接搜尋 Gemini，進入 https://gemini.google.com，再用申請的教育信箱登入。
  • 身分確認：檢查右上角頭像，確保顯示教育帳號。

三、Nano Banana 2：五個實作

實作一

日常轉正：生活照變沙龍照 / 證件照

實戰流程：

  1. 提供基準人像（上傳生活照）。
  2. 在對話工具箱裡選擇香蕉圖示【建立圖像】。
  3. 直接貼上提示詞：
    • 韓系沙龍照：請把這張生活照轉成韓系沙龍照，背景乾淨簡約、光線柔和，穿米色針織上衣，整體質感自然精緻。
    • 專業證件照：請把這張生活照轉成正式證件照，穿白襯衫加深藍西裝，灰色背景，正面看鏡頭。

實作三

製作宣傳圖卡：從對話直出視覺設計

實戰流程：

  1. 在對話工具箱裡選擇香蕉圖示【建立圖像】。
  2. 直接在對話裡說明要產生宣傳圖卡並說明內容。
  3. 範例指令：產生一張宣傳圖卡：週三研習 2026-AI must-knows 梯次1（3月4日下午1:40-3:00）行政人員及資優、潛能班老師；梯次2（3月11日下午1:40-3:00）其餘人員；地點：電腦教室

AI 自動生成的研習宣傳圖卡 🔍

Tip 1：AI 能自動抓取文字重點並進行視覺排版，不需要自己手動排版。
Tip 2：也可省略步驟一（工具箱選擇香蕉圖示），但提示詞必須明確提到類似「產生圖片」的字眼。
Tip 3：新改版的 Nano Banana 2 使用 Flash 模型即可處理繁體中文字。

四、Vibe Coding：說話即程式

互動遊戲製作（Canvas）

核心：宇宙探險：公轉自轉與晝夜互動教材。

流程：Canvas 模式下指令：「產生適合國小學生的認識公轉自轉互動介面」。

老師只需思考教學邏輯，以前要學半年的動畫，現在 30 秒搞定。

行政一頁式網頁

核心：將繁瑣 PDF 流程轉化為易操作的數位工具。

流程：上傳新生入學流程 PDF，指令做成 HTML 網頁方便人員瀏覽。

Google Sites 部署

流程：協作平台建立空白網站 → 插入「整頁嵌入」 → 貼上 HTML 代碼 → 設為首頁並發布。

不需要租伺服器，利用教育帳號就能快速打造數位導航站。

五、NotebookLM：處理資訊超載

全自動產出：廣播劇與簡報大綱

流程：建立筆記本 → 新增 PDF 來源 → 「工作室 (Studio)」生成「語音摘要」。

與其看幾十頁報告，不如讓 AI 講給你聽。

Lyria 3 音樂創作

流程：「創作音樂」模式 → 文字下咒或圖片導向作曲。

製作影片背景音樂或校園活動主題曲的神器。
