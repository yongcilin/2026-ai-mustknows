四、Vibe Coding：說話即程式

互動遊戲製作 —— 國小四年級：電路串聯與並聯

核心概念：運用 Gemini Canvas 功能，不需要寫一行程式碼，直接透過對話描述需求，生成可運行的互動式網頁教材。

實戰流程示範：

  1. 切換工具：在 Gemini 選擇 Canvas 模式。
  2. 一句話下咒 (Vibe Coding)：請在 Canvas 中為我製作一個『國小四年級：電路串聯與並聯』的互動教學（html）
  3. AI 自動開發：
    • 視覺建模：AI 自動使用 HTML/CSS/JS 建立電池、電線與燈泡的模擬元件。
    • 互動設計：加入開關與接線功能，讓學生觀察不同接法下燈泡亮度的變化。

【成品】學生可手動操作的「電路串聯與並聯」模擬互動教材 🔍

這就是 Vibe Coding 的威力。老師只需要思考「教學邏輯」，AI 負責「程式實作」。以前要外包或學半年的動畫，現在 30 秒就搞定。

打造校務行政「入學諮詢一頁式網頁」

核心概念：將繁瑣 PDF 流程轉化為易操作的數位工具。

實戰流程示範：

  1. 提供素材：上傳校內 PDF 📄。
  2. 一句話下咒 (Vibe Coding)：這是一個供校內同仁接到入學詢問時可以對照答詢的流程，我想要做成網頁供人員方便瀏覽（包含手機），HTML 格式。
  3. AI 自動開發：自動擷取 PDF 關鍵步驟，生成包含分類標籤與摺疊視窗的現代化網頁。

【成品預覽】入學諮詢一頁式網頁 🔗

1. 記得註明是 HTML 格式方便未來部署。
2. Gemini 若沒自動開啟 Canvas 來 Vibe coding，請記得工具箱手動點選。

部署上線 —— 使用 Google 協作平台發佈你的 AI 網頁

核心概念：將 Vibe Coding 產生的 HTML 代碼，透過 Google 協作平台（Google Sites）轉化為正式且公開的網址，達成真正的工作落地。

實戰部署流程：

  1. 進入平台：Gmail 右上角九宮格選 「協作平台」，點選 「+ 空白網站」。
  2. 整頁嵌入：
    • 右側切換至 「頁面」 分頁。
    • 滑鼠移至下方 「+」 符號，選取彈出的 「整頁嵌入」。
    • 輸入名稱（如：入學電話答詢系統）。
  3. 植入代碼：
    • 點選畫面中央 「新增嵌入內容」 → 「嵌入程式碼」。
    • 貼上：將剛才 Gemini Canvas 產生的 HTML 完整代碼貼入。（剛剛的 Canvas 代碼區右上角有一個共用按鍵，點下去就有複製內容的選項）
    • 點選 「下一個」 → 「插入」。
  4. 頁面優化：
    • 在右側「頁面」清單，將「入學電話答詢系統」點選三個點 「設為首頁」。
    • 將原本預設的舊「首頁」點選三個點 「刪除」。
  5. 發佈與分享：
    • 上方工具列 「與他人共用」：將「已發佈的網站」設為 「公開」 並完成。
    • 點選上方 「發布」：輸入網址結尾（如：enrollment-qa），再次按 「發布」。
    • 再次點選「與他人共用」旁的連結圖示，即可 「複製已發佈的網站連結」。
1. 這套流程不需要租伺服器、不需要買網域。利用教育版帳號，你能隨時為任何行政流程快速打造專屬的「數位導航站」。
2. 註：Pro（付費版）用戶可直接分享 Canvas 內容，但教育版或免費帳戶目前只能手動以此方法發佈。

五、NotebookLM：一鍵產生海報、簡報、廣播劇的好利器

一鍵自動產出：語音摘要 影片摘要 心智圖 報告 學習卡 測驗 資訊圖表 簡報 資料表

核心概念：NotebookLM 具備強大的「長文本解析 + 內容結構化」能力，能將生硬的計畫書或教材，瞬間轉化為動聽的對談與視覺架構。

實戰流程示範：

  1. 進入平台：九宮格選 NotebookLM（或直接搜尋進入）。
  2. 建立筆記本：點選「建立新的筆記本」。
  3. 新增來源：左側點選「新增來源」，將想要做成多媒體的 計畫 PDF、書商教案 PDF 或 相關網址 加進來。 📥
  4. AI 摘要解析：讀取後，畫面中間的對話窗格會自動摘要所有來源內容。
  5. 生成多媒體：在右側 「工作室 (Studio)」 選取要生成的內容，例如 「語音摘要 (AI 廣播劇)」。

NotebookLM 成果預覽 🔍

一個主題（例如滿清政府下的台灣治理）可上傳多個 PDF 或網址，NotebookLM 會自己幫你閱讀過所有文件再摘要。

六、創意魔法：音樂與影像

Lyria 3 音樂創作 —— 老師也能當製作人

核心概念：Google 最新的 Lyria 3 模型能理解情感與情境，只需透過對話或「圖片」，就能在幾秒鐘內產出具備專業編曲水準的音樂。

實戰流程示範：

  1. 選取工具：在 Gemini 工具箱（或側邊欄）點選 「創作音樂」 模式。
  2. 創意下咒 (兩種方式)：
    • 文字導向：輸入提示詞，例如：幫我產生一首適合國小健康促進的歌，曲風要活潑、有朝氣，帶點輕快的鼓點。
    • 視覺導向：直接上傳一張照片（如：學生在操場奔跑的照片），請 AI 根據圖片氛圍創作對應的背景音樂。
  3. AI 自動生成：Lyria 3 會自動完成作曲、編曲與樂器配置，產出高品質音軌。
這在製作「影片背景音樂」或是「校園活動主題曲」時超級好用。告別版權爭議，用 AI 量身打造專屬樂章。

Grok 4.2 創意實作 —— 讓圖片動起來的影像魔法

帳號啟用流程：

  1. 開啟 grok.com，點選 Continue with Google 快速登入。
  2. 選取要綁定的 Google 帳號並完成授權。

產生動態影像：

  1. 登入 Grok 後，左側點選 Imagine。
  2. 在下方輸入欄位點選 「+ 照片」 圖示，上傳想要動態化的照片。
  3. 在欄位右側選擇 「製作影片」。
  4. 點選送出，等待系統生成動態影像。
1. 製作影片時，先挑「主體清楚、背景不太亂」的照片，成功率更高、效果也更自然。
2. 可延伸應用到校史老照片、地方文史教材的動態化教學。
